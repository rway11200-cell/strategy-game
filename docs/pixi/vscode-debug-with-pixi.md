# 🐞 Ejecutar PixiJS **solo en modo Debug** (Vite + VS Code)

Esta guía explica **cómo ejecutar el proyecto únicamente en modo debug**, usando **breakpoints de VS Code** (círculos rojos) y **Chrome controlado por VS Code**.

---

## 📌 Requisitos

- Proyecto **PixiJS** con **Vite**
- VS Code
- Navegador **Chrome**
- El proyecto **NO se abre directamente en el navegador**
- El proyecto **SIEMPRE se inicia con `npm run dev`**

---

## 1️⃣ Iniciar el servidor de desarrollo

Desde la raíz del proyecto:

```bash
npm run dev
```

El servidor queda corriendo en:
<http://localhost:8080>

y ya para en los puntos de interrupcion
